1. The Goal

Core concept

You design the contact/landing page on GitHub Pages ‚ûú then embed that page into RENTMY using a simple <iframe> with a fixed height, so:

2. HTML You Paste into RENTMY

In the RENTMY page editor, switch to the HTML view and paste this. Replace YOUR_GITHUB_URL_HERE with your actual GitHub page URL. 

<iframe 
  src="YOUR_GITHUB_URL_HERE" 
  width="100%" 
  height="2000px" 
  title="GitHub Project" 
  style="border: none; overflow: hidden;"
  scrolling="no">
</iframe>

The important pieces:

  • width="100%" ‚Üí iframe stretches full width of the content area.
  • height="2000px" ‚Üí enough vertical space so the whole GitHub page fits.
  • style="border: none; overflow: hidden;" ‚Üí no border + no inner scrollbars.
  • scrolling="no" ‚Üí tells the browser not to show scrollbars inside the frame.
Adjusting the height

If your page is taller, bump the height up (2200px, 2500px, etc.) until the inner scrollbar disappears and only the main page scrolls.

3. What RENTMY Changes It To

After you save the page, RENTMY rewrites your HTML and wraps the iframe. It ends up looking like this: 

<figure>
  <div class="embed-content">
    <iframe 
      src="https://gabrielafiona.github.io/bougiebeachpartyco/contact" 
      width="100%" 
      height="1700px" 
      title="GitHub Project" 
      style="border: none; overflow: hidden;" 
      scrolling="no">
    </iframe>
  </div>
</figure>

Key things RENTMY does:

  • Wraps the iframe in a <figure> element.
  • Adds a wrapper <div class="embed-content"> around it.
  • Keeps the important attributes: width, height, style, scrolling.

4. Why This Fixes the Double-Scroll Problem

  1. Normally, if the content inside the iframe is taller than the iframe itself, the browser adds a scrollbar inside the iframe.
  2. At the same time, the main RENTMY page also has its own scrollbar. That‚Äôs where the ‚Äúscrolling in a box inside a page‚Äù feeling comes from.
  3. By making the iframe‚Äôs height large enough and using overflow: hidden; + scrolling="no", the inner scrollbar disappears.
  4. Now the GitHub page is just a big, tall section of the RENTMY page. You scroll once, like a normal long page. ‚ú®
